## Description

This project is a responsive student portfolio website created using HTML, CSS and JavaScript.

The website contains:

- Home Page
- About Me
- Projects
- Academic Planner
- Contact Page

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Responsive Design
- Navigation Menu
- Academic Planner
- Contact Form Validation
- Local Storage
- Embedded Multimedia

## How to Run

1. Download the project.
2. Open the folder in Visual Studio Code.
3. Open `index.html` in your browser.
